## Statement

Resonance exists to become the shared foundation for spatial interaction in multiplayer and simulation software.
It should let any engine adopt the same conceptual model without inheriting engine-specific coupling.

## What Success Looks Like

- A single vocabulary for spatial communication primitives.
- Clean separation between core concepts and engine SDKs.
- Long-lived documentation that remains understandable years after initial delivery.
