## Index

## Primary Use Cases

- Spatial voice communication in multiplayer games.
- Proximity-based interaction zones in 3D worlds.
- Engine-independent communication channels for social spaces.
- Simulation environments that need spatial presence without engine lock-in.

## Secondary Use Cases

- Tooling and editor integrations that visualize communication boundaries.
- Reference implementations for SDK authors.
- Benchmarks and validation harnesses for future performance work.
